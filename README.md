# FitLife
FitLife is a mobile application that tracks users’ daily physical activities — including steps, running, calories, water intake, and sleep — and provides personalized recommendations. The app helps users build healthy habits by monitoring their progress, setting fitness goals, and giving real-time motivational feedback.


Key Features:

* Step, distance, and calorie tracking using device sensors
* Daily water intake reminders and progress tracking
* Personalized Recommendations and insights
* Interactive charts for weekly and monthly performance
* Local data storage (SQLite)
* Smart notifications and goal reminders

Fitlife will be released on Google Play soon.
